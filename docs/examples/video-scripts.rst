Video Script Taslaklari
=======================

Asagidaki metinler, pazarlama ve onboarding videolari icin kisa script taslaklaridir.

1) 60 saniyelik urun tanitimi
-----------------------------

Sahne 1 (0-10sn):
- "ArchiFlow ile diskinizdeki karmasayi guvenli sekilde duzenleyin."

Sahne 2 (10-25sn):
- Kaynak klasor secimi
- `Preview` butonu ile risk almadan analiz

Sahne 3 (25-45sn):
- Kopya gruplarinin listelenmesi
- Quarantine modunun guvenli calisma mantigi

Sahne 4 (45-60sn):
- Ozet karti + Apply onayi
- "Yanlislik olursa transaction ile Undo destegi"

2) 3 dakikalik hizli baslangic videosu
--------------------------------------

Bolum 1:
- Kurulum ve ilk acilis

Bolum 2:
- Preview -> Dry-run -> Apply akisi

Bolum 3:
- Quarantine klasorunu acma ve geri alma

Bolum 4:
- CSV/JSON rapor export

3) 5 dakikalik guvenlik odakli video
------------------------------------

Basliklar:
1. Neden silme default degil?
2. Transaction journaling nasil calisir?
3. Cancel/crash sonrasi geri donus senaryosu
4. Buyuk disklerde adim adim guvenli kullanim
