Katki Rehberi
=============

Katki sureci ve gelistirme standartlari icin proje kokundeki ``CONTRIBUTING.md`` dosyasina bakin.

Ozet:

1. Yeni branch acin.
2. Degisiklikleri kucuk ve test edilebilir parcalarda yapin.
3. Kod kalitesi kontrollerini calistirin.
4. Acik, izlenebilir bir aciklama ile pull request acin.
