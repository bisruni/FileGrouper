Baslangic Rehberi
=================

1. Sanal ortam olusturun ve aktif edin.
2. Bagimliliklari yukleyin.
3. GUI ile onizleme calistirin.

.. code-block:: bash

   python3 -m venv .venv
   source .venv/bin/activate
   python3 -m pip install -r requirements.txt
   python3 main.py gui

Ilk guvenli calisma akisi:

1. Kaynak klasoru secin.
2. ``Onizleme`` calistirin.
3. Ozet kartini kontrol edin.
4. Gerekirse test modundan cikarak uygulayin.
